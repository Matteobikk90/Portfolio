Code Hunt

Code Hunt The home page shows posts from all users, and each users profile shows only that users posts. A user can register for an account which allows them to add posts. Users can add a category and language to their post, and posts can then be filtered and viewed by a specific category or language. Users can also like other users posts.

Tech:

Mongo, AJAX, Passport, NodeJS, JavaScript, jQuery, HTML5, CSS3, Bootstrap
